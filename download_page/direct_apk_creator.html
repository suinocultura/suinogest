Método Online (Recomendado)

Este método não requer instalação de ferramentas de desenvolvimento e pode ser usado diretamente de qualquer dispositivo com acesso à internet.

Nota: O APK gerado será enviado para seu e-mail após a criação.

Método Android Studio

Para desenvolvedores que desejam personalizar ainda mais o aplicativo.

  1. Baixe o projeto base: Baixar Projeto Android
  2. Abra o projeto no Android Studio:

    Importe o projeto baixado usando a opção "Import Project" no Android Studio.

  3. Personalize conforme necessário:

    Você pode modificar ícones, cores, temas e outras configurações no projeto.

  4. Compile o APK:

    Use a opção "Build > Build Bundle(s) / APK(s) > Build APK(s)" no Android Studio.

Importante: Este método requer conhecimentos básicos de desenvolvimento Android.

Perguntas Frequentes

O aplicativo funciona offline?

O aplicativo requer conexão com a internet para acessar o Sistema Suinocultura. Porém, ele implementa cache para melhorar a performance e permitir navegação limitada offline.

Como instalar o APK no Android?

Após baixar o APK, toque nele para instalá-lo. Você precisará permitir a instalação de "Fontes desconhecidas" nas configurações de segurança do seu dispositivo.

Este método é melhor que o método Kivy?

Este método é mais confiável e funciona em mais dispositivos que o método Kivy/Buildozer. Ele é recomendado para a maioria dos usuários devido à sua simplicidade e compatibilidade.
